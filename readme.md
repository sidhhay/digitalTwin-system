# Digital Twin Diabetes Management System

## Overview

This project is a DBMS-based healthcare system that models a digital twin for diabetic patients. It stores patient data, monitors glucose levels, and generates risk assessments and alerts.

##  Features

* Patient data management
* Glucose monitoring (time-series)
* Lifestyle and medication tracking
* Risk assessment system
* Alert generation for high-risk patients

##  Database Design

* ER Diagram included
* Normalized schema (3NF)
* Relationships: 1:N, M:N (resolved using bridge table)

##  Technologies

* Oracle SQL / PL-SQL
* DBMS concepts (DDL, DML, Constraints)

## Future Work

* Trigger-based alert automation
* Backend API integration
* Frontend dashboard
